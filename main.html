현장 요청 등록, 인력 관리, 현장 배정, 배치내역 확인, 근무시간 입력, 정산까지 필요한 페이지로 바로 이동할 수 있습니다.

5 주요 페이지
전화번호 매칭 배정과 작업자 계정 연결

PAGE 1

현장 배정

현장 요청 등록, 요청 확인, 현장 배정과 배정 상태를 관리합니다.

PAGE 2

인력관리

관리자만 인력을 추가하거나 삭제하고 현재 등록된 인력 목록을 확인합니다.

PAGE 3

배치내역

현장별 배치 상태와 실제 배정된 인력 목록을 확인합니다.

PAGE 4

근무시간 입력

작업자 계정으로 로그인한 인력이 자신의 배정 내역과 근무시간을 입력합니다.

PAGE 5

정산

근무시간과 공수 데이터를 기준으로 현장 정산 내용을 확인합니다.

PAGE 6

관리자 승인

관리자 가입 신청 목록을 확인하고 승인 상태를 처리합니다.

# 인력 배정 운영센터

거래처 요청 접수, 인력 배정, 현장 출퇴근 체크, 거래처별 출퇴근 데이터 조회를 한 화면에서 처리할 수 있는 정적 웹앱입니다.

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. 거래처 요청을 등록합니다.
3. 인력을 추가하고 현장에 배정합니다.
4. 출근/퇴근 체크를 누르면 데이터가 누적됩니다.
5. `CSV 다운로드` 버튼으로 거래처별 출퇴근 데이터를 저장할 수 있습니다.

## 포함 기능

- 문자/전화로 받은 거래처 요청 등록
- 보유 인력 목록 관리
- 요청별 다중 인력 배정
- 출근 / 퇴근 체크 시간 저장
- 거래처별 출퇴근 데이터 필터링
- CSV 내보내기

## 저장 방식

브라우저 `localStorage`를 사용해 같은 브라우저에서는 데이터가 유지됩니다.

# 인력 배정 운영센터

거래처 요청 접수, 인력 배정, 현장 출퇴근 체크, 거래처별 출퇴근 데이터 조회를 한 화면에서 처리할 수 있는 정적 웹앱입니다.

## 실행 방법

1. `index.html`을 브라우저에서 열고 로그인합니다.
2. 관리자 계정은 `main.html`에서 인력배정과 정산 페이지로 이동합니다.
3. 사용자 계정은 `worker-entry.html`에서 근무시간을 입력합니다.
4. `settlement.html`에서 거래처별 근무시간과 공수를 확인합니다.
5. `CSV 다운로드` 버튼으로 정산 데이터를 저장할 수 있습니다.

## 포함 기능

- 문자/전화로 받은 거래처 요청 등록
- 보유 인력 목록 관리
- 요청별 다중 인력 배정
- 출근 / 퇴근 체크 시간 저장
- 거래처별 출퇴근 데이터 필터링
- CSV 내보내기

## 저장 방식

브라우저 `localStorage`를 사용해 같은 브라우저에서는 데이터가 유지됩니다.
